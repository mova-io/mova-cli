# Tiers and Pricing

## Free tier

100 API requests per day, 1 user, community support only, no SLA.
Hard-capped at the daily limit; requests fail with HTTP 429 once
the cap is hit. No overage; no upgrade-on-overage path. Intended
for hobbyists, evaluation, and OSS contributors.

## Pro tier

$29/month per seat. Unlimited API requests within the soft 5000/day
cap (overage at $0.0001 per request, optional monthly ceiling).
Includes 10 integrations, email + chat support, 99.9% uptime SLA.
Supports OAuth and password + 2FA; SAML SSO is enterprise-only.

## Enterprise tier

Custom pricing, contact sales. Unlimited API requests with
negotiated commitments (typically 1M-10M requests/month, overage
at $0.00005/request). Includes SAML SSO, SCIM provisioning, audit
logs with 1-year retention, dedicated customer success manager,
99.95% uptime SLA, custom data processing agreement, multi-region
data residency, and IP allowlisting.

## Custom domain support

Pro tier supports custom logo + accent color override. Enterprise
tier adds full white-label: custom domain (CNAME), custom logo,
custom color palette, custom email-sender domain, and custom
favicon. Enterprise setup requires a 5-business-day solutions
engineering engagement.
